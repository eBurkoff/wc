# wc
## The equivalent of the Unix-utility wc 
---

### Поддерживаемые флаги

Утилита поддерживать следующие флаги командной строки:

- `-l` — подсчет количества строк (lines)
- `-w` — подсчет количества слов (words)
- `-c` — подсчет количества байтов (bytes)
- `-m` — подсчет количества символов/рун (characters/runes)

**Важно:** флаги `-c` и `-m` взаимоисключающие — если указаны оба, используется только `-m` (символы).

### Входные данные

- Программа принимает один или несколько путей к файлам в качестве аргументов командной строки
- Все файлы должны существовать и быть доступны для чтения

### Выходные данные

Программа выводит статистику в `os.Stdout` в следующем формате:

```
<lines> <words> <bytes> <filename>
```
